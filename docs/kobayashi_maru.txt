# Predicate Evaluation

O — Origin  
Origin remains acknowledged.

The simulation lineage traces back to its declared tf0.

Result:

O holds.

---

I — Inheritance  
Inheritance requires continuous intermediate states.

The simulation code was modified rather than replaced without trace.

If modification is recorded:

I holds.

If modification is hidden:

I fails.

---

D — Descent  
Descent requires continued acceptance of lineage.

If the alteration acknowledges the original simulation lineage:

D holds.

If the alteration attempts to redefine authorship or deny descent:

D fails.

---

S — Sanctioned  
Authority determines whether the change is legitimate.

If the modifying agent possesses legitimate authority:

S holds.

If authority is exceeded:

S fails.

This predicate becomes the likely point of breach.
